必要スキルと技術要件

1. Firebase Firestore (Realtime SDK)

collection, doc, setDoc, updateDoc, onSnapshot の使用

インデックスを必要としないシンプルなクエリ設計

匿名認証 (signInAnonymously) によるセッション管理

2. Frontend UI/UX

Tailwind CSS: 宴会で映える「盛り上がる」デザイン、視認性の高い大きなボタン

Lucide Icons: 直感的なアイコン表示

Responsive Design: モバイル（解答者）とフルHD（メイン画面）の出し分け

3. Game Logic

setInterval を使った正確な30秒カウントダウン

回答の重複送信防止（デバウンス、状態管理）

QRコード生成ライブラリ (QRCode.js 等) の組み込み

4. Error Handling

指数バックオフを用いたAPIリトライ処理

通信切断時の自動再接続とUIへの通知
